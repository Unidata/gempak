     CENTER

     CENTER allows the GD2NDFD user to specify the originating or generating
     center ID and sub-center ID, as well as the Generating Process/Model ID.
     The input is given as follows:

                Center ID #/Sub-Center ID #/Generating ID #

     Values may be omitted in specifying CENTER.  For example, to omit
     the first value, specify only the second, enter the following:

		/Sub-Center ID #               


     If values are omitted, the following defaults are inserted:

                ID #  Default value           Meaning

               Center        7         Center # (7->NCEP)
               Sub           5         Sub-center (5->HPC)

     See NCEP OFFICE NOTE 388 for more information on how to set these
     parameters.  Specifically, ON388 - Table 0 for Originating Center, ON388 
     - Table C for Originating Sub-center, and ON388 - Table A for Generating
