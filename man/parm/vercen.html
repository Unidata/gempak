     VERCEN

     VERCEN allows the GDGRIB user to specify the contents of bytes 4,
     5, 6, and 26 of the GRIB PDS.  The input is given as follows:

                byte_4/byte_5/byte_6/byte_26

     Values may be omitted in specifying VERCEN.  For example, to omit
     the first and third values, specifying only the second and fourth,
     enter the following:

               /byte_5//byte_26


     If values are omitted, the following defaults are inserted:

               BYTE    Default value           Meaning

                 4           2         Parameter Table Version #
                 5           7         CENTER # (7->NCEP)
                 6           0         Generating Process ID
                26           5         Sub-center (5->HPC)
