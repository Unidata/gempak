Enhancement Tables

Satellite enhancements can be used in Gempak programs as well as NSAT. Enhancement tables are found under the directory $GEMTBL/luts.

Several predefined color tables are provided with the Gempak distribution, and can be used within Gempak programs by specifying: LUT enhancement.tbl at the program prompt after the display is generated.

The default enhancement used for each image type is defined in the file IMGTYP.TBL. To select a different default enhancement for a given satellite image, simply enter a new enhancement name under the DEFAULT LUT FILE column.

Adding New Enhancement Tables

If you want to add new enhancement tables of your own, be sure to follow the format of one of the existing files. Location of RGB values is dependent on column location. When new enhancements are created, you will need to add the name to the $NAWIPS/tables/nsat/enhance.tbl file so that they will be available in the NSAT enhancements pulldown menu. Note, the enhancement names must end with the .tbl extension.
