AVN model Display

The image was generated using GDCNTR and the Northern Hemisphere Topographic areafile included with the Gempak5.2.1/NAWIPS distribution. AREA files including GOES-8 and GOES-9 images are specified with the SATFIL variable, and PROJ = sat. NIDS and NOWrad radar images may be used as backdrops using the RADFIL variable, and PROJ = rad.

Example Script 

#!/bin/csh -f

gdcntr << EOF
GDATTIM  = f000
GLEVEL   = 0
GVCORD   = PRES
GFUNC    = pmsl
GDFILE   = $HDS/96012012_thin.gem
CINT     = 4
LINE     = 1/1/2/2
MAP      = 32
TITLE    = 1/-2/AVN Surface Pressure ~
DEVICE   = xw
SATFIL   = $GEMPAKHOME/areas/topo_nh.area
RADFIL   = 
PROJ     = sat
GAREA    = 15;-130;35;-15
CLEAR    = y
PANEL    = 0
TEXT     = 1.2/23/1/hw
SCALE    = 999
LATLON   = 0
HILO     = 4;2/H;L
HLSYM    = 9;1/2/33;2//hw
CLRBAR   = 
CONTUR   = 3/3
SKIP     = 0
FINT     = 
FLINE    = 
CTYPE    = C
r

e
EOF
exit(0)
Composite Images

Topographic bases can be composited with satellite imagery in MCIDAS to produce complex image bases. Here is an MPEG annimation of hourly sea level pressure data over infrared satellite imagery and the topographic base map. Surface observations are decoded with SATOGEM, and objectively analyzed with OABSFC . GOES 8 infrared imagery is thresholded at -10C.
