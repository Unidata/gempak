Grid point cross section

An interactive choice of cross section endpoints can be accomplished with GDCNTR and GDCROSS.

After using GDCNTR to plot the sea level pressure map, use the CURSOR cxstns command to select the cross section axis using the mouse. The CXSTNS axis is stored in the nts file variable. Then, run GDCROSS using the stored CXSTNS value.
