Blizzard of 1996 NOWrad

The WSI NOWrad image was generated using GPMAP. GEMPAK programs read the WSI INC NOWrad images in their native format. All programs that support the RADFIL variable option can be used to overlay data.

MAP       = 5
GAREA     = dset
PROJ      = sat
SATFIL    = $NOWRAD/NOW_960121_2100
RADFIL    =           
LATLON    = 0
PANEL     = 0
TITLE     = 1/-2
TEXT      = 1.2/23/1/hw
CLEAR     = y
DEVICE    = xw
GEMPAK-GPMAP>r
GPWATCH can be used to plot watch boxes decoded with the watchtogem decoder.

 MAP      = 6/1/1
 GAREA    = uslcc
 PROJ     = rad
 SATFIL   = 
 RADFIL   = $NOWRAD/NOW_960613_2100
 LATLON   = 0
 PANEL    = 0
 TITLE    = 3/-1/Watches 
 TEXT     = 1/22/1/hw
 CLEAR    = yes
 DEVICE   = xw
 WWFIL    = $WATCHES/watches_1996.gem
 WWS      = cur
 WWATT    = y
 GEMPAK-GPWATCH>
