NSAT display

The WSI NIDS image was generated using NSAT. NAST provide a Graphical User Interface to Satellite and Radar imagery. Pull down menus and Pop up widgets allow the user to animate images, select from an array of color enhancements, and automatically uppdate loops as new files are received. For more information on setting up the directory structure in NSAT see the discussion on Image data directories.
