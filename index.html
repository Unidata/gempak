GEMPAK Documentation

Note: The GEMPAK software distribution is no longer supported by Unidata. As of early 2021, development and support of the package have been taken up by a group of community members who will manage the GEMPAK Github repository.

The GEMPAK documentation provided here represents the state of the package as of version 7.5.1, its last release by Unidata (February 2019).

How To Get GEMPAK Support

Didn't find a solution in the GEMPAK documentation? Unidata's resources for supporting GEMPAK are very limited, but we can suggest the following:

  • The GEMBUD Mailing List Archive where you read discussion among members of the Unidata community or subscribe to the list.
  • The Support Question Archive where you can questions discussed in Unidata's support system.
  • The project's Github Issues page may prove useful.
